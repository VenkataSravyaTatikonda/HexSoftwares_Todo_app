TO-DO List Web Application
This is a simple and responsive To-Do list web application built using HTML,CSS and JavaScript. The application allows users to efficiently manage their daily-tasks by adding new tasks, marking htem as completed,editing tasks,and removing tasks when they are no longer needed.
The app uses the browser localStorage to store tasks,ensuring that all data is preserved even after refreshing or closing the browser. It features a clean and user-friendly interface that makes task managemnet easy and intuitive.

Features
  * Add new tasks
  * Edit existing tasks
  * Mark tasks as completed
  * Remove tasks
  * Data persistence using local storage
  * Responsive and clean UI

Technologies Used
  * HTML
  * CSS
  * JavaScript
